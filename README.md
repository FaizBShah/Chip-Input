# Chip-Input

### An input with chips autocomplete and dropdown support

## Steps to run the code

```
npm install
npm run dev
```

## Steps to build the code

```
npm install
npm run build
```

The server will then start on `localhost:3000`.

Node version required - `20.10.0`

NPM version required - `10.2.5`
